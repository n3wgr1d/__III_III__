Project Description:

The goal of this project was to test my artistic project on the real environment but still on a very small scale.

The concept behind it was to create a immersive audio music performance with songs that were produced by me and my colleague.

First test was conducted at home with 1st order Ambisonics(IEM). You can read more about it here.

Audio Setup:

6 SOUNDBOKS Speakers; RME Fireface UCX; 4DSOUND software.
